Key Features

Secure Authentication

Ensures that only authorized users can access their notes. Uses JWT (JSON Web Tokens) for secure user authentication and session management.

Note Organization

Allows users to categorize and tag their notes for easy retrieval. Users can organize notes by categories, tags, and custom labels.

Advanced Search

Provides powerful search capabilities to find notes quickly. Users can search by keywords, tags, or full-text content within notes.

About Note Keeper API

This project was inspired by a personal need to organize notes efficiently. The idea grew during my time in School, where I learned the importance of managing information effectively. This is a Portfolio Project for Holberton School. You can find the project repository here.
